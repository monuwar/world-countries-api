OVERVIEW

Welcome to the World Countries API. A comprehensive, lightweight, open-source static dataset containing fully-mapped details for over 240 countries and territories.

This includes ISO alpha codes, capital cities, calling codes, currency schema, local spoken languages, standard flags, and customized Telegram Premium custom animated emoji mapping.

Powered entirely by globally-replicated static CDN. Highly cached, CORS-enabled by default, with absolute zero rate limits. Perfect for production microservices.

INTEGRATION EXAMPLES

Python (Requests SDK) 
import requests

url = "https://cdn.jsdelivr.net/gh/monuwar/world-countries-api@main/countries.json"
response = requests.get(url)

if response.status_code == 200:
    countries = response.json()
    for country in countries[:3]: 
        print(f"Name: {country['name']} | Emoji ID: {country['telegram_emojis']['classic_id']}")
Java (Native HttpClient) 
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class Main {
    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://cdn.jsdelivr.net/gh/monuwar/world-countries-api@main/countries.json"))
            .build();

        client.sendAsync(request, HttpResponse.BodyHandlers.ofString())
            .thenApply(HttpResponse::body)
            .thenAccept(System.out::println)
            .join();
    }
}
